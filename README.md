# Pringle Solutions Website

Welcome to the Pringle Solutions website repository! This is a professional website designed for a family business offering computer services, professional photography, and household maintenance.

## 🎨 Design Features

- **Blue & White Theme**: Professional color scheme with:
  - Primary Blue: #1e3a5f
  - Light Blue: #2c5aa0
  - Accent Blue: #3b82f6
  - White & Light Gray backgrounds

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional interface with smooth animations and transitions
- **Easy Navigation**: Sticky navigation bar with smooth scrolling

1. **Customize Contact Information**
   - Update phone number in the Contact section
   - Replace `info@pringlesolutions.co.za` with your actual email
   - Update location information

2. **Add Your Photos**
   - Replace placeholder portfolio images with actual work samples
   - Update service icons if needed

3. **Update Statistics**
   - Modify the "Happy Clients", "Years Experience", and other stats in the About section

4. **Add Social Media Links**
   - Update the social media URLs in the footer

5. **Deploy the Website**
   - Push this repository to GitHub Pages or your hosting provider

## 🔧 Customization Guide

### Colors
To change the color scheme, modify the CSS variables in `styles.css`:
```css
:root {
    --primary-blue: #1e3a5f;
    --light-blue: #2c5aa0;
    --accent-blue: #3b82f6;
    --white: #ffffff;
    --light-gray: #f5f7fa;
    --dark-gray: #4a5568;
    --text-color: #2d3748;
}
```

### Fonts
The website uses 'Segoe UI' as the primary font. To change it, modify:
```css
body {
    font-family: 'Your Font Here', sans-serif;
}
```

### Content
All text content is in `index.html`. Simply edit the HTML to update:
- Service descriptions
- About information
- Contact details
- Any other text

## 📧 Contact Form

The contact form is functional on the frontend but requires backend setup to actually send emails. To enable email functionality:

1. Use a service like Formspree, Netlify Forms, or a custom backend
2. Update the form's `action` attribute or add form submission handling
